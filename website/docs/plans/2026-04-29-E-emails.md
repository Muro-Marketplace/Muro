> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans. Steps use checkbox (`- [ ]`) syntax for tracking. Linked spec: `docs/specs/2026-04-29-batch-fixes-design.md` § Workstream E.

**Goal:** Customer order emails render the artwork image. Artist names show as display names (not slugs or IDs). Em dashes removed from user-facing email strings.

**Architecture:** A small `resolve-artist-name` helper does slug-shape detection and a Supabase lookup, with a bulk variant for webhook callers. The Stripe webhook persists `image` URL in the `orders.items` JSONB so post-write emails (shipping, delivery confirmations) can read deterministically. Three customer-facing order email templates are wired to use the existing shared `<OrderSummary />` component which already renders `<Img>`. Em-dash sweep targets all `.tsx` and `.ts` files under `src/emails/`.

**Tech Stack:** Resend · `@react-email/components` · Next.js 16 (App Router) · Supabase · TypeScript · Vitest.

**Dependencies:**
- Workstream C does NOT block this; the `image` and `artistName` fixes are independent.
- Touches the same Stripe webhook as workstream C; commit C first to minimise merge conflicts.

**Items covered:** 9 (artwork images), 10 (artist names), 16 (overall email review). Item 11 (em-dash sweep) is partially handled here (emails) and completed in workstream F (rest of site).

- [ ] **Step 1: Helper**

```ts
// src/emails/_helpers/resolve-artist-name.ts
import type { SupabaseClient } from "@supabase/supabase-js";

const SLUG_RE = /^[a-z0-9]+(?:-[a-z0-9]+)*$/;

export function looksLikeSlug(s: string | null | undefined): boolean {
  if (!s) return false;
  return SLUG_RE.test(s.trim());
}

export async function resolveArtistName(
  db: SupabaseClient,
  slug: string | null | undefined,
  fallbackName?: string | null,
): Promise<string> {
  if (fallbackName && !looksLikeSlug(fallbackName)) return fallbackName.trim();
  if (!slug) return fallbackName?.trim() || "Artist";
  const { data } = await db
    .from("artist_profiles")
    .select("name")
    .eq("slug", slug)
    .maybeSingle();
  return data?.name?.trim() || fallbackName?.trim() || slug;
}

export async function resolveArtistNamesBulk(
  db: SupabaseClient,
  slugs: Array<string | null | undefined>,
): Promise<Map<string, string>> {
  const cleaned = Array.from(new Set(slugs.filter((s): s is string => !!s)));
  if (cleaned.length === 0) return new Map();
  const { data } = await db
    .from("artist_profiles")
    .select("slug, name")
    .in("slug", cleaned);
  const map = new Map<string, string>();
  for (const row of data || []) {
    if (row.slug && row.name) map.set(row.slug, row.name);
  }
  return map;
}
```
